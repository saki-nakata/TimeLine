## 共通レイアウト（認証済み画面）

S-03〜S-08 はすべて共通 `Layout` コンポーネントを使用する。
レイアウトはブレークポイントに応じて3段階で変化する。

### デスクトップ（lg: 1024px 以上）

```
+--------------------+------------------------------------------+
|  TimeLine          |                                          |
|                    |  [画面コンテンツ]                         |
|  🏠 ホーム         |                                          |
|  👥 フォロー中     |                                          |
|  🔍 検索           |                                          |
|                    |                                          |
|  [投稿する]        |                                          |
|                    |                                          |
|  [アバター]        |                                          |
|  表示名            |                                          |
|  @username         |                                          |
|  [ログアウト]      |                                          |
+--------------------+------------------------------------------+
```

| 要素 | 内容 |
|------|------|
| TimeLine ロゴ（テキスト） | クリックで `/home` へ遷移 |
| ホーム | `/home` へ遷移。アクティブ時は `bg-gray-200` |
| フォロー中 | `/following` へ遷移。アクティブ時は `bg-gray-200` |
| 検索 | クリックで SearchOverlay（インライン検索バー）をトグル |
| 投稿する（テキストボタン） | 新規投稿モーダルを開く |
| ユーザーカード | アバター・表示名・@username を表示。クリックで `/profile/{id}` へ遷移 |
| ログアウト | ログアウトして S-01 へ遷移 |

### タブレット（md: 768px 以上 〜 lg 未満）

左サイドバーを幅 64px のアイコンのみ表示に縮小する。

```
+------+------------------------------------------+
|  T   |                                          |
|      |  [画面コンテンツ]                         |
|  🏠  |                                          |
|  👥  |                                          |
|  🔍  |                                          |
|      |                                          |
|  ✏️  |                                          |
|      |                                          |
| [Av] |                                          |
+------+------------------------------------------+
```

| 要素 | 内容 |
|------|------|
| T ロゴ | 「T」のみ表示。クリックで `/home` へ遷移 |
| ナビゲーション | アイコンのみ（テキストなし） |
| 投稿ボタン | 鉛筆アイコン（丸ボタン） |
| アバター | クリックでドロップダウンメニュー（プロフィール / ログアウト）を表示 |

### スマートフォン（md: 768px 未満）

左サイドバーを非表示にし、上部ヘッダー＋下部ボトムナビゲーションに切り替える。

```
+------------------------------------------+
|              TimeLine                    |  ← 上部固定ヘッダー（検索バー非表示時）
+------------------------------------------+
|                                          |
|  [画面コンテンツ]                         |
|                                          |
|                                          |
+------------------------------------------+
| 🏠  |  👥  |  🔍  |  ✏️  |  [Av] |     |  ← 下部固定ナビゲーション
+------------------------------------------+
```

検索アイコンタップ時は上部ヘッダーが SearchOverlay に切り替わる：

```
+------------------------------------------+
| [🔍 ユーザーを検索...          ] [×]     |  ← SearchOverlay（sticky）
+------------------------------------------+
|  ┌────────────────────────────────────┐  |
|  │ @alice                             │  |  ← インラインドロップダウン結果
|  │ @alice_works                       │  |
|  └────────────────────────────────────┘  |
|  [画面コンテンツ]                         |
+------------------------------------------+
```

プロフィールアイコンタップ時は右下にポップアップメニューを表示：

```
+------------------------------------------+
|                                          |
|                      +----------------+  |
|                      |  プロフィール  |  |
|                      |  ログアウト    |  |
| 🏠  |  👥  |  🔍  |  ✏️  | +----------------+
+------------------------------------------+
```

| 要素 | 内容 |
|------|------|
| 上部ヘッダー | 「TimeLine」ロゴを中央表示（sticky）。SearchOverlay 表示中は非表示 |
| ボトムナビゲーション | 画面下部に固定。ホーム / フォロー中 / 検索 / 投稿ボタン / プロフィールアイコン |
| 検索アイコン | タップで SearchOverlay をトグル |
| 投稿ボタン | 丸型の青いボタン（鉛筆アイコン） |
| プロフィールアイコン | タップで右下にポップアップメニューを表示（プロフィール / ログアウト）|

## S-01: ログイン画面

認証されていないユーザーがログインする画面。未認証状態でのアクセス先として `/login` にリダイレクトする。

```
+--------------------------------------------------+
|                   TimeLine                       |
+--------------------------------------------------+
|                                                  |
|              ┌──────────────────────────┐        |
|              │ メールアドレス            │        |
|              └──────────────────────────┘        |
|              ┌──────────────────────────┐        |
|              │ パスワード               │        |
|              └──────────────────────────┘        |
|                                                  |
|              [      ログイン      ]              |
|                                                  |
|        アカウントをお持ちでない方はこちら          |
|              → [新規登録]                        |
|                                                  |
+--------------------------------------------------+
```

### 画面仕様

| 要素 | 内容 |
|------|------|
| メールアドレス入力 | type="email"、必須 |
| パスワード入力 | type="password"、必須 |
| ログインボタン | クリックで `/api/auth/login` を呼び出す |
| エラー表示 | 認証失敗時にフォーム上部にエラーメッセージを表示 |
| 新規登録リンク | `/register` へ遷移 |

## S-03: タイムライン（ホーム）画面

ログイン後のメイン画面。全ユーザーの投稿またはフォロー中ユーザーの投稿をタブで切り替えて表示する。左サイドバーは共通 Layout を使用（「共通レイアウト」を参照）。

```
+--------------------+------------------------------------------+
|  TimeLine          |  [ 全て ]  [ フォロー中 ]                |
|                    |  ──────────────────────────────────────  |
|  🏠 ホーム ◀      |                                          |
|  👥 フォロー中     |  ┌──────────────────────────────────┐   |
|  🔍 ユーザー検索   |  │ 投稿内容を入力...（最大 280 文字）│   |
|                    |  │ [📎 画像を追加]       [投 稿]    │   |
|  [投稿する]        |  └──────────────────────────────────┘   |
|                    |                                          |
|  [アバター]        |  ┌──────────────────────────────────┐   |
|  @username         |  │ [アバター] @username · 3 分前    │   |
|  [ログアウト]      |  │ 投稿テキストの内容がここに表示   │   |
+--------------------+  │ ♥ 12  [いいね]  💬 4  [コメント] │   |
                        └──────────────────────────────────┘   |
                                                               |
                        ┌──────────────────────────────────┐   |
                        │ [アバター] @another · 1 時間前   │   |
                        │ 別の投稿テキストの内容です。      │   |
                        │ ♥ 3  [いいね]   💬 1  [コメント] │   |
                        └──────────────────────────────────┘   |
                                                               |
                                   [ もっと読み込む ]           |
                        +------------------------------------------+
```

### 画面仕様

| 要素 | 内容 |
|------|------|
| サイドバー | 共通 Layout（TimeLine ロゴ・ナビゲーション・ログアウト） |
| 投稿フォーム | テキストエリア（最大 280 文字）・画像添付ボタン・投稿ボタン |
| 文字数カウンター | 入力文字数 / 280 をリアルタイム表示 |
| タブ | 「全て」「フォロー中」を切り替え。選択中のタブを下線で強調 |
| 投稿カード | @username・投稿日時・テキスト・画像（任意）・いいね数・コメント数 |
| @username | クリックでプロフィール画面（S-05）へ遷移 |
| [いいね] | クリックでいいね/取り消しをトグル。自分のいいね済みは色付きで表示 |
| [コメント] | クリックで投稿詳細画面（S-04）へ遷移 |
| 投稿カード全体 | クリックで投稿詳細画面（S-04）へ遷移 |
| もっと読み込む | 追加投稿を取得（ページネーション） |

### 画面仕様

| 要素 | 内容 |
|------|------|

| [削除] | 確認ダイアログ後に `DELETE /api/posts/{id}` を呼び出す |
| コメント一覧 | 時系列順（古い順）で全コメントを表示 |
| [削除（コメント）] | 自分のコメントのみ表示。即時削除 |
| コメント入力 | テキストエリア・コメントボタン |

## S-06: ユーザー検索（SearchOverlay）

専用ページではなく、各画面のコンテンツ上部にインラインで表示される検索 UI。
ナビゲーションの「検索」アイコン（サイドバー / ボトムナビ）をクリック/タップすることで表示・非表示をトグルする。
ページ遷移時は自動的に閉じる。

```
+------------------------------------------------------------------+
|  [🔍 ユーザーを検索...                    ] [×閉じる]           |
+------------------------------------------------------------------+
|                                                                  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  [アバター]  表示名                                        │  |  ← インライン
|  │              @alice                                       │  |     ドロップダウン
|  └────────────────────────────────────────────────────────────┘  |     （入力後表示）
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  [アバター]  表示名                                        │  |
|  │              @alice_works                                  │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|  [画面コンテンツ（後ろに続く）]                                   |
|                                                                  |
+------------------------------------------------------------------+
```

### 画面仕様

| 要素 | 内容 |
|------|------|
| 検索バー | sticky 表示。入力欄・クリアボタン・閉じるボタンを含む |
| 検索条件 | username・display_name の両方に対して部分一致（ILIKE）検索 |
| デバウンス | 入力から 300ms 後に `GET /api/users/search?q=xxx` を呼び出す |
| ドロップダウン結果 | ヒットしたユーザーのアバター・表示名・@username を一覧表示（最大 20 件） |
| ユーザー選択 | クリックで SearchOverlay を閉じてプロフィール画面（S-05）へ遷移 |
| 0 件時 | 「見つかりませんでした」を表示 |
| 閉じる | [×] ボタンまたは Esc キーで閉じる |

## S-07: プロフィール編集モーダル

自分のアイコン画像・表示名・ユーザー名・自己紹介文を編集するモーダル。プロフィール画面（S-05）の「プロフィールを編集」ボタンで開く。独立した URL はなく、S-05 上にオーバーレイとして表示される。

```
  ┌────────────────────────────────────────────────────────────┐
  │  プロフィールを編集                            [× 閉じる]  │
  │  ────────────────────────────────────────────────────────  │
  │                                                            │
  │  アイコン画像                                              │
  │  ┌──────────┐  [画像を変更]                               │
  │  │ [現在の  │                                              │
  │  │  アイコン│                                              │
  │  └──────────┘                                              │
  │                                                            │
  │  表示名                                                    │
  │  ┌──────────────────────────────────────────────────────┐  │
  │  │ 現在の表示名                                          │  │
  │  └──────────────────────────────────────────────────────┘  │
  │                                                            │
  │  ユーザー名（@username）                                   │
  │  ┌──────────────────────────────────────────────────────┐  │
  │  │ 現在のユーザー名                                      │  │
  │  └──────────────────────────────────────────────────────┘  │
  │                                                            │
  │  自己紹介                                                  │
  │  ┌──────────────────────────────────────────────────────┐  │
  │  │ 現在の自己紹介文が入力済みで表示される               │  │
  │  └──────────────────────────────────────────────────────┘  │
  │                                                            │
  │                       [キャンセル]   [  保  存  ]          │
  └────────────────────────────────────────────────────────────┘
```

### 画面仕様

| 要素 | 内容 |
|------|------|
| [× 閉じる] | モーダルを閉じて S-05 に戻る（変更破棄） |
| アイコン画像 | 現在のアイコンをプレビュー表示。「画像を変更」で新しい画像を選択 |
| 画像アップロード | 対応フォーマット: JPEG / PNG / WebP / GIF、最大 5MB。開発: ローカルファイルシステム、本番: AWS S3 |
| 表示名テキストボックス | 現在の displayName をプリセット表示。任意入力（最大 100 文字） |
| ユーザー名テキストボックス | 現在の username をプリセット表示。半角英数字・アンダースコア、最大 50 文字 |
| 自己紹介テキストエリア | 現在の bio をプリセット表示。任意入力 |
| [保存] | `PUT /api/users/{id}` (displayName / bio) + `POST /api/users/{id}/avatar` (画像変更時) を呼び出し、成功後モーダルを閉じて S-05 を更新 |
| [キャンセル] | 変更を破棄してモーダルを閉じる |

## S-08: フォロー中一覧画面

ログインユーザーがフォローしているユーザーの一覧を表示する画面。左サイドバーの「フォロー中」から遷移する。

+--------------------+------------------------------------------+

### 画面仕様

| 要素 | 内容 |
|------|------|
| サイドバー | 共通 Layout（「フォロー中」がアクティブ表示） |
| ユーザーカード | アバター・表示名・@username・bio を表示 |
| @username / アバター | クリックでそのユーザーのプロフィール画面（S-05）へ遷移 |
| [フォロー中] ボタン | クリックでアンフォロー。一覧からリアルタイムに削除される |
| 空状態 | フォロー中のユーザーがいない場合は「フォロー中のユーザーはいません」を表示 |
| データ取得 | `GET /api/users/{currentUserId}/following` |
