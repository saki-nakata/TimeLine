## S-01: ログイン画面

認証されていないユーザーがログインする画面。未認証状態でのアクセス先として `/login` にリダイレクトする。

```
+--------------------------------------------------+
|                   TimeLine                       |
+--------------------------------------------------+
|                                                  |
|              ┌──────────────────────────┐        |
|              │ メールアドレス            │        |
|              └──────────────────────────┘        |
|              ┌──────────────────────────┐        |
|              │ パスワード               │        |
|              └──────────────────────────┘        |
|                                                  |
|              [      ログイン      ]              |
|                                                  |
|        アカウントをお持ちでない方はこちら          |
|              → [新規登録]                        |
|                                                  |
+--------------------------------------------------+
```

### 画面仕様

| 要素 | 内容 |
|------|------|
| メールアドレス入力 | type="email"、必須 |
| パスワード入力 | type="password"、必須 |
| ログインボタン | クリックで `/api/auth/login` を呼び出す |
| エラー表示 | 認証失敗時にフォーム上部にエラーメッセージを表示 |
| 新規登録リンク | `/register` へ遷移 |

## S-02: 新規登録画面

新規ユーザーがアカウントを作成する画面。

```
+--------------------------------------------------+
|                   TimeLine                       |
+--------------------------------------------------+
|                                                  |
|              ┌──────────────────────────┐        |
|              │ ユーザー名（@username）   │        |
|              └──────────────────────────┘        |
|              ┌──────────────────────────┐        |
|              │ メールアドレス            │        |
|              └──────────────────────────┘        |
|              ┌──────────────────────────┐        |
|              │ パスワード               │        |
|              └──────────────────────────┘        |
|              ┌──────────────────────────┐        |
|              │ パスワード（確認）        │        |
|              └──────────────────────────┘        |
|                                                  |
|              [      登  録      ]                |
|                                                  |
|         すでにアカウントをお持ちの方はこちら       |
|              → [ログイン]                        |
|                                                  |
+--------------------------------------------------+
```

### 画面仕様

| 要素 | 内容 |
|------|------|
| ユーザー名入力 | 半角英数字・アンダースコア、最大 50 文字、必須 |
| メールアドレス入力 | type="email"、必須 |
| パスワード入力 | 8 文字以上、必須 |
| パスワード（確認）入力 | パスワードと一致するか確認 |
| 登録ボタン | クリックで `/api/auth/register` を呼び出し、成功後 `/` にリダイレクト |
| エラー表示 | バリデーション失敗・重複時にエラーメッセージを表示 |
| ログインリンク | `/login` へ遷移 |

## S-03: タイムライン（ホーム）画面

ログイン後のメイン画面。全ユーザーの投稿またはフォロー中ユーザーの投稿をタブで切り替えて表示する。

```
+------------------------------------------------------------------+
|  TimeLine          [🔍 ユーザー検索]  [@username]  [ログアウト]  |
+------------------------------------------------------------------+
|                                                                  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │ 投稿内容を入力...（最大 280 文字）                          │  |
|  │                                                            │  |
|  │ [📎 画像を追加]                         [投 稿]           │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|  [ 全て ]  [ フォロー中 ]                                        |
|  ─────────────────────────────────────────────────────────────   |
|                                                                  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  @username · 3 分前                                        │  |
|  │  投稿テキストの内容がここに表示されます。                  │  |
|  │  [添付画像（あれば表示）]                                  │  |
|  │  ─────────────────────────────────────────────────────    │  |
|  │  ♥ 12  [いいね]    💬 4  [コメント]                       │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  @another_user · 1 時間前                                  │  |
|  │  別の投稿テキストの内容です。                              │  |
|  │  ─────────────────────────────────────────────────────    │  |
|  │  ♥ 3   [いいね]    💬 1  [コメント]                       │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|                    [ もっと読み込む ]                            |
+------------------------------------------------------------------+
```

### 画面仕様

| 要素 | 内容 |
|------|------|
| ヘッダー | ロゴ・ユーザー検索ボタン・ログインユーザー名・ログアウトボタン |
| 投稿フォーム | テキストエリア（最大 280 文字）・画像添付ボタン・投稿ボタン |
| 文字数カウンター | 入力文字数 / 280 をリアルタイム表示 |
| タブ | 「全て」「フォロー中」を切り替え。選択中のタブを下線で強調 |
| 投稿カード | @username・投稿日時・テキスト・画像（任意）・いいね数・コメント数 |
| @username | クリックでプロフィール画面（S-05）へ遷移 |
| [いいね] | クリックでいいね/取り消しをトグル。自分のいいね済みは色付きで表示 |
| [コメント] | クリックで投稿詳細画面（S-04）へ遷移 |
| 投稿カード全体 | クリックで投稿詳細画面（S-04）へ遷移 |
| もっと読み込む | 追加投稿を取得（ページネーション） |

## S-04: 投稿詳細画面

投稿の全文とコメント一覧を表示する画面。

```
+------------------------------------------------------------------+
|  TimeLine          [🔍 ユーザー検索]  [@username]  [ログアウト]  |
+------------------------------------------------------------------+
|  ← タイムラインに戻る                                            |
|                                                                  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  @username · 2026-05-17 09:00                              │  |
|  │  投稿の全文テキストがここに表示されます。                  │  |
|  │  [添付画像（あれば表示）]                                  │  |
|  │  ─────────────────────────────────────────────────────    │  |
|  │  ♥ 12  [いいね]    💬 4 件のコメント    [🗑 削除※]        │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|  コメント（4 件）                                                 |
|  ─────────────────────────────────────────────────────────────   |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  @commenter · 10 分前                                      │  |
|  │  コメントの内容がここに表示されます。     [🗑 削除※]       │  |
|  └────────────────────────────────────────────────────────────┘  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  @another · 5 分前                                         │  |
|  │  別のコメントの内容です。                 [🗑 削除※]       │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  コメントを入力...                                          │  |
|  └────────────────────────────────────────────────────────────┘  |
|  [   コメントする   ]                                            |
|                                                                  |
+------------------------------------------------------------------+
```

※ 削除ボタンは自分の投稿・コメントのみ表示

### 画面仕様

| 要素 | 内容 |
|------|------|
| 戻るリンク | タイムライン（S-03）または元の画面に戻る |
| 投稿カード | 投稿全文・画像・いいね数・コメント数 |
| [いいね] | クリックでいいね/取り消しをトグル |
| [削除（投稿）] | 自分の投稿のみ表示。確認後に削除 |
| コメント一覧 | 時系列順（古い順）で全コメントを表示 |
| [削除（コメント）] | 自分のコメントのみ表示。即時削除 |
| コメント入力 | テキストエリア・コメントボタン |

## S-05: プロフィール画面

ユーザーのプロフィールと投稿一覧を表示する画面。他ユーザーのプロフィールを見てフォローできる。

```
+------------------------------------------------------------------+
|  TimeLine          [🔍 ユーザー検索]  [@username]  [ログアウト]  |
+------------------------------------------------------------------+
|                                                                  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  [アバター]  @username                                     │  |
|  │              フォロワー: 128  フォロー中: 45               │  |
|  │              登録日: 2026-01-01                            │  |
|  │                                  [フォローする / フォロー中] │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|  @username の投稿（12 件）                                        |
|  ─────────────────────────────────────────────────────────────   |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  投稿テキスト...                                           │  |
|  │  ♥ 5   [いいね]    💬 2  [コメント]                       │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  別の投稿テキスト...                                        │  |
|  │  ♥ 1   [いいね]    💬 0  [コメント]                       │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|                    [ もっと読み込む ]                            |
+------------------------------------------------------------------+
```

### 画面仕様

| 要素 | 内容 |
|------|------|
| アバター | アバター画像（未設定時はデフォルト画像） |
| @username | ユーザー名 |
| フォロワー / フォロー中 | それぞれの数を表示 |
| 登録日 | アカウント作成日 |
| [フォローする / フォロー中] | 他ユーザーのプロフィール時のみ表示。自分のプロフィールには非表示。フォロー済みの場合は「フォロー中」と表示し、クリックでアンフォロー |
| 投稿一覧 | そのユーザーの投稿を新着順で表示 |

## S-06: ユーザー検索結果画面

ユーザー名で検索した結果を一覧表示する画面。

```
+------------------------------------------------------------------+
|  TimeLine          [🔍 ユーザー検索]  [@username]  [ログアウト]  |
+------------------------------------------------------------------+
|                                                                  |
|  ┌──────────────────────────────────────────┐  [検索]           |
|  │ ユーザー名を入力...                       │                   |
|  └──────────────────────────────────────────┘                   |
|                                                                  |
|  「alice」の検索結果（3 件）                                      |
|  ─────────────────────────────────────────────────────────────   |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  [アバター]  @alice                                        │  |
|  │              フォロワー: 42  フォロー中: 18  [フォローする] │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  [アバター]  @alice_works                                  │  |
|  │              フォロワー: 10  フォロー中: 5   [フォロー中]  │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  [アバター]  @alice123                                     │  |
|  │              フォロワー: 3   フォロー中: 2   [フォローする] │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
+------------------------------------------------------------------+
```

### 画面仕様

| 要素 | 内容 |
|------|------|
| 検索入力フォーム | ユーザー名（@username）で部分一致検索 |
| 検索ボタン | クリックで `/api/users/search?q=xxx` を呼び出す |
| 結果一覧 | ヒットしたユーザーのアバター・@username・フォロワー数・フォロー中数・フォローボタンを表示 |
| @username | クリックでプロフィール画面（S-05）へ遷移 |
| [フォローする / フォロー中] | フォロー状態に応じてボタンラベルを切り替え |
| 0 件時 | 「該当するユーザーが見つかりません」を表示 |
