# TimeLine 画面設計書

**バージョン:** 1.2
**作成日:** 2026-05-17
**作成者:** Nakata Saki

---

## 画面一覧

| 画面ID | 画面名 | URL | 認証要否 |
|--------|--------|-----|---------|
| S-01 | ログイン | `/login` | 不要 |
| S-02 | 新規登録 | `/register` | 不要 |
| S-03 | タイムライン（ホーム） | `/home` | 必要 |
| S-04 | 投稿詳細 | `/posts/:id` | 必要 |
| S-05 | プロフィール | `/profile/:userId` | 必要 |
| S-06 | ユーザー検索結果 | `/search?q=xxx` | 必要 |
| S-07 | プロフィール編集 | （モーダル） | 必要 |
| S-08 | フォロー中一覧 | `/following` | 必要 |

## 共通レイアウト（認証済み画面）

S-03〜S-08 はすべて左サイドバー付きの共通 `Layout` コンポーネントを使用する。

```
+--------------------+------------------------------------------+
|  TimeLine          |                                          |
|                    |  [画面コンテンツ]                         |
|  🏠 ホーム         |                                          |
|  👥 フォロー中     |                                          |
|  🔍 ユーザー検索   |                                          |
|                    |                                          |
|  [投稿する]        |                                          |
|                    |                                          |
|  [アバター]        |                                          |
|  @username         |                                          |
|  [ログアウト]      |                                          |
+--------------------+------------------------------------------+
```

| 要素 | 内容 |
|------|------|
| TimeLine ロゴ | クリックで `/home` へ遷移 |
| ホーム | `/home` へ遷移。現在のページが `/home` の場合は `bg-gray-200` でハイライト |
| フォロー中 | `/following` へ遷移。現在のページが `/following` の場合は `bg-gray-200` でハイライト |
| ユーザー検索 | 検索モーダルを開く（または `/search` へ遷移） |
| 投稿する | 新規投稿モーダルを開く（S-03 のみ機能する）|
| ユーザーカード | アバター・@username を表示。自分のプロフィール画面が `/profile/{自分のId}` の場合は `bg-gray-200` でハイライト |
| ログアウト | ログアウトして S-01 へ遷移 |

## S-01: ログイン画面

認証されていないユーザーがログインする画面。未認証状態でのアクセス先として `/login` にリダイレクトする。

```
+--------------------------------------------------+
|                   TimeLine                       |
+--------------------------------------------------+
|                                                  |
|              ┌──────────────────────────┐        |
|              │ メールアドレス            │        |
|              └──────────────────────────┘        |
|              ┌──────────────────────────┐        |
|              │ パスワード               │        |
|              └──────────────────────────┘        |
|                                                  |
|              [      ログイン      ]              |
|                                                  |
|        アカウントをお持ちでない方はこちら          |
|              → [新規登録]                        |
|                                                  |
+--------------------------------------------------+
```

### 画面仕様

| 要素 | 内容 |
|------|------|
| メールアドレス入力 | type="email"、必須 |
| パスワード入力 | type="password"、必須 |
| ログインボタン | クリックで `/api/auth/login` を呼び出す |
| エラー表示 | 認証失敗時にフォーム上部にエラーメッセージを表示 |
| 新規登録リンク | `/register` へ遷移 |

## S-02: 新規登録画面

新規ユーザーがアカウントを作成する画面。

```
+--------------------------------------------------+
|                   TimeLine                       |
+--------------------------------------------------+
|                                                  |
|              ┌──────────────────────────┐        |
|              │ ユーザー名（@username）   │        |
|              └──────────────────────────┘        |
|              ┌──────────────────────────┐        |
|              │ メールアドレス            │        |
|              └──────────────────────────┘        |
|              ┌──────────────────────────┐        |
|              │ パスワード               │        |
|              └──────────────────────────┘        |
|              ┌──────────────────────────┐        |
|              │ パスワード（確認）        │        |
|              └──────────────────────────┘        |
|                                                  |
|              [      登  録      ]                |
|                                                  |
|         すでにアカウントをお持ちの方はこちら       |
|              → [ログイン]                        |
|                                                  |
+--------------------------------------------------+
```

### 画面仕様

| 要素 | 内容 |
|------|------|
| ユーザー名入力 | 半角英数字・アンダースコア、最大 50 文字、必須 |
| メールアドレス入力 | type="email"、必須 |
| パスワード入力 | 8 文字以上、必須 |
| パスワード（確認）入力 | パスワードと一致するか確認 |
| 登録ボタン | クリックで `/api/auth/register` を呼び出し、成功後 `/` にリダイレクト |
| エラー表示 | バリデーション失敗・重複時にエラーメッセージを表示 |
| ログインリンク | `/login` へ遷移 |

## S-03: タイムライン（ホーム）画面

ログイン後のメイン画面。全ユーザーの投稿またはフォロー中ユーザーの投稿をタブで切り替えて表示する。左サイドバーは共通 Layout を使用（「共通レイアウト」を参照）。

```
+--------------------+------------------------------------------+
|  TimeLine          |  [ 全て ]  [ フォロー中 ]                |
|                    |  ──────────────────────────────────────  |
|  🏠 ホーム ◀      |                                          |
|  👥 フォロー中     |  ┌──────────────────────────────────┐   |
|  🔍 ユーザー検索   |  │ 投稿内容を入力...（最大 280 文字）│   |
|                    |  │ [📎 画像を追加]       [投 稿]    │   |
|  [投稿する]        |  └──────────────────────────────────┘   |
|                    |                                          |
|  [アバター]        |  ┌──────────────────────────────────┐   |
|  @username         |  │ [アバター] @username · 3 分前    │   |
|  [ログアウト]      |  │ 投稿テキストの内容がここに表示   │   |
+--------------------+  │ ♥ 12  [いいね]  💬 4  [コメント] │   |
                        └──────────────────────────────────┘   |
                                                               |
                        ┌──────────────────────────────────┐   |
                        │ [アバター] @another · 1 時間前   │   |
                        │ 別の投稿テキストの内容です。      │   |
                        │ ♥ 3  [いいね]   💬 1  [コメント] │   |
                        └──────────────────────────────────┘   |
                                                               |
                                   [ もっと読み込む ]           |
                        +------------------------------------------+
```

### 画面仕様

| 要素 | 内容 |
|------|------|
| サイドバー | 共通 Layout（TimeLine ロゴ・ナビゲーション・ログアウト） |
| 投稿フォーム | テキストエリア（最大 280 文字）・画像添付ボタン・投稿ボタン |
| 文字数カウンター | 入力文字数 / 280 をリアルタイム表示 |
| タブ | 「全て」「フォロー中」を切り替え。選択中のタブを下線で強調 |
| 投稿カード | @username・投稿日時・テキスト・画像（任意）・いいね数・コメント数 |
| @username | クリックでプロフィール画面（S-05）へ遷移 |
| [いいね] | クリックでいいね/取り消しをトグル。自分のいいね済みは色付きで表示 |
| [コメント] | クリックで投稿詳細画面（S-04）へ遷移 |
| 投稿カード全体 | クリックで投稿詳細画面（S-04）へ遷移 |
| もっと読み込む | 追加投稿を取得（ページネーション） |

## S-04: 投稿詳細画面

投稿の全文とコメント一覧を表示する画面。

```
+------------------------------------------------------------------+
|  TimeLine          [🔍 ユーザー検索]  [@username]  [ログアウト]  |
+------------------------------------------------------------------+
|  ← タイムラインに戻る                                            |
|                                                                  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  @username · 2026-05-17 09:00                              │  |
|  │  投稿の全文テキストがここに表示されます。                  │  |
|  │  [添付画像（あれば表示）]                                  │  |
|  │  ─────────────────────────────────────────────────────    │  |
|  │  ♥ 12  [いいね]    💬 4 件のコメント         [… ※]         │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|  ※ 「…」メニュー（自分の投稿のみ表示）                            |
|  +─────────────+                                                 |
|  │ ✏ 編集      │                                                 |
|  │ 🗑 削除     │                                                 |
|  +─────────────+                                                 |
|                                                                  |
|  ── 投稿編集モーダル（「編集」選択時）──────────────────────────  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  投稿を編集                                    [× 閉じる]  │  |
|  │  ┌──────────────────────────────────────────────────────┐  │  |
|  │  │ 既存のテキストが入力済みで表示される（最大 280 文字）  │  │  |
|  │  └──────────────────────────────────────────────────────┘  │  |
|  │  [既存画像プレビュー / 画像変更ボタン]                      │  |
|  │                        [キャンセル]  [保 存]                │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|  コメント（4 件）                                                 |
|  ─────────────────────────────────────────────────────────────   |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  @commenter · 10 分前                                      │  |
|  │  コメントの内容がここに表示されます。     [🗑 削除※]       │  |
|  └────────────────────────────────────────────────────────────┘  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  @another · 5 分前                                         │  |
|  │  別のコメントの内容です。                 [🗑 削除※]       │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  コメントを入力...                                          │  |
|  └────────────────────────────────────────────────────────────┘  |
|  [   コメントする   ]                                            |
|                                                                  |
+------------------------------------------------------------------+
```

※ 「…」ボタン・編集モーダルは自分の投稿のみ表示

### 画面仕様

| 要素 | 内容 |
|------|------|
| 戻るリンク | タイムライン（S-03）または元の画面に戻る |
| 投稿カード | 投稿全文・画像・いいね数・コメント数 |
| [いいね] | クリックでいいね/取り消しをトグル |
| [… メニュー] | 自分の投稿のみ表示。「編集」「削除」を選択できる |
| 編集モーダル | テキスト・画像を変更して「保存」で `PUT /api/posts/{id}` を呼び出す |
| [削除] | 確認ダイアログ後に `DELETE /api/posts/{id}` を呼び出す |
| コメント一覧 | 時系列順（古い順）で全コメントを表示 |
| [削除（コメント）] | 自分のコメントのみ表示。即時削除 |
| コメント入力 | テキストエリア・コメントボタン |

## S-05: プロフィール画面

ユーザーのプロフィールと投稿一覧を表示する画面。他ユーザーのプロフィールを見てフォローできる。

```
+------------------------------------------------------------------+
|  TimeLine          [🔍 ユーザー検索]  [@username]  [ログアウト]  |
+------------------------------------------------------------------+
|                                                                  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  [アバター]  @username                                     │  |
|  │              フォロワー: 128  フォロー中: 45               │  |
|  │              Web開発が好きです。（自己紹介 / bio）          │  |
|  │              [プロフィールを編集 ※自分のみ]                │  |
|  │                                  [フォローする / フォロー中] │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|  @username の投稿（12 件）                                        |
|  ─────────────────────────────────────────────────────────────   |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  投稿テキスト...                                           │  |
|  │  ♥ 5   [いいね]    💬 2  [コメント]                       │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|  ┌────────────────────────────────────────────────────────────┐  |
|  │  別の投稿テキスト...                                        │  |
|  │  ♥ 1   [いいね]    💬 0  [コメント]                       │  |
|  └────────────────────────────────────────────────────────────┘  |
|                                                                  |
|                    [ もっと読み込む ]                            |
+------------------------------------------------------------------+
```

### 画面仕様

| 要素 | 内容 |
|------|------|
| アバター | アバター画像（未設定時はデフォルト画像） |
| @username | ユーザー名 |
| フォロワー / フォロー中 | それぞれの数を表示 |
| 自己紹介（bio） | 設定済みの場合に表示。未設定時は非表示 |
| [プロフィールを編集] | 自分のプロフィール画面のみ表示。クリックでプロフィール編集モーダル（S-07）を開く |
| [フォローする / フォロー中] | 他ユーザーのプロフィール時のみ表示。自分のプロフィールには非表示。フォロー済みの場合は「フォロー中」と表示し、クリックでアンフォロー |
| 投稿一覧 | そのユーザーの投稿を新着順で表示 |

## S-07: プロフィール編集モーダル

自分のアイコン画像・表示名・ユーザー名・自己紹介文を編集するモーダル。プロフィール画面（S-05）の「プロフィールを編集」ボタンで開く。独立した URL はなく、S-05 上にオーバーレイとして表示される。

```
  ┌────────────────────────────────────────────────────────────┐
  │  プロフィールを編集                            [× 閉じる]  │
  │  ────────────────────────────────────────────────────────  │
  │                                                            │
  │  アイコン画像                                              │
  │  ┌──────────┐  [画像を変更]                               │
  │  │ [現在の  │                                              │
  │  │  アイコン│                                              │
  │  └──────────┘                                              │
  │                                                            │
  │  表示名                                                    │
  │  ┌──────────────────────────────────────────────────────┐  │
  │  │ 現在の表示名                                          │  │
  │  └──────────────────────────────────────────────────────┘  │
  │                                                            │
  │  ユーザー名（@username）                                   │
  │  ┌──────────────────────────────────────────────────────┐  │
  │  │ 現在のユーザー名                                      │  │
  │  └──────────────────────────────────────────────────────┘  │
  │                                                            │
  │  自己紹介                                                  │
  │  ┌──────────────────────────────────────────────────────┐  │
  │  │ 現在の自己紹介文が入力済みで表示される               │  │
  │  └──────────────────────────────────────────────────────┘  │
  │                                                            │
  │                       [キャンセル]   [  保  存  ]          │
  └────────────────────────────────────────────────────────────┘
```

### 画面仕様

| 要素 | 内容 |
|------|------|
| [× 閉じる] | モーダルを閉じて S-05 に戻る（変更破棄） |
| アイコン画像 | 現在のアイコンをプレビュー表示。「画像を変更」で新しい画像を選択 |
| 画像アップロード | 対応フォーマット: JPEG / PNG / WebP / GIF、最大 5MB。開発: ローカルファイルシステム、本番: AWS S3 |
| 表示名テキストボックス | 現在の displayName をプリセット表示。任意入力（最大 100 文字） |
| ユーザー名テキストボックス | 現在の username をプリセット表示。半角英数字・アンダースコア、最大 50 文字 |
| 自己紹介テキストエリア | 現在の bio をプリセット表示。任意入力 |
| [保存] | `PUT /api/users/{id}` (displayName / bio) + `POST /api/users/{id}/avatar` (画像変更時) を呼び出し、成功後モーダルを閉じて S-05 を更新 |
| [キャンセル] | 変更を破棄してモーダルを閉じる |

## S-08: フォロー中一覧画面

ログインユーザーがフォローしているユーザーの一覧を表示する画面。左サイドバーの「フォロー中」から遷移する。

```
+--------------------+------------------------------------------+
|  TimeLine          |  フォロー中                              |
|                    |  ──────────────────────────────────────  |
|  🏠 ホーム         |                                          |
|  👥 フォロー中 ◀  |  ┌──────────────────────────────────┐   |
|  🔍 ユーザー検索   |  │ [アバター] @alice                │   |
|                    |  │  Alice                           │   |
|  [投稿する]        |  │  自己紹介テキスト...              │   |
|                    |  │                  [フォロー中]    │   |
|  [アバター]        |  └──────────────────────────────────┘   |
|  @username         |                                          |
|  [ログアウト]      |  ┌──────────────────────────────────┐   |
+--------------------+  │ [アバター] @bob                  │   |
                        │  Bob                             │   |
                        │  別の自己紹介...                 │   |
                        │                  [フォロー中]    │   |
                        └──────────────────────────────────┘   |
                        +------------------------------------------+
```

### 画面仕様

| 要素 | 内容 |
|------|------|
| サイドバー | 共通 Layout（「フォロー中」がアクティブ表示） |
| ユーザーカード | アバター・表示名・@username・bio を表示 |
| @username / アバター | クリックでそのユーザーのプロフィール画面（S-05）へ遷移 |
| [フォロー中] ボタン | クリックでアンフォロー。一覧からリアルタイムに削除される |
| 空状態 | フォロー中のユーザーがいない場合は「フォロー中のユーザーはいません」を表示 |
| データ取得 | `GET /api/users/{currentUserId}/following` |
