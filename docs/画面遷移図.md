# TimeLine 画面遷移図

**バージョン:** 1.1
**作成日:** 2026-05-17
**作成者:** Nakata Saki

---

## 画面遷移図

```mermaid
graph TD
    START([アクセス開始]) --> AUTH{認証済み？}

    AUTH -- 未認証 --> S01[S-01: ログイン画面]
    AUTH -- 認証済み --> S03[S-03: タイムライン /home]

    S01 -- ログイン成功 --> S03
    S01 -- 「新規登録」クリック --> S02[S-02: 新規登録画面]

    S02 -- 登録成功 --> S03
    S02 -- 「ログイン」クリック --> S01

    %% 共通サイドバー遷移（S-03〜S-08 すべてから可能）
    SIDEBAR_HOME[サイドバー: TimeLine / ホーム] -- クリック --> S03
    SIDEBAR_FOLLOWING[サイドバー: フォロー中] -- クリック --> S08[S-08: フォロー中一覧 /following]
    SIDEBAR_LOGOUT[サイドバー: ログアウト] -- クリック --> S01

    S03 -- 投稿カードクリック --> S04[S-04: 投稿詳細画面]
    S03 -- 「コメント」ボタンクリック --> S04
    S03 -- @username クリック --> S05[S-05: プロフィール画面]
    S03 -- 「ユーザー検索」クリック --> S06[S-06: ユーザー検索結果画面]

    S04 -- 「← 戻る」クリック --> S03
    S04 -- @username クリック --> S05

    S05 -- 「← 戻る」クリック --> S03
    S05 -- 投稿カードクリック --> S04
    S05 -- 「プロフィールを編集」クリック --> S07_MODAL[S-07: プロフィール編集モーダル\n※ S-05 上にオーバーレイ表示]

    S06 -- @username クリック --> S05
    S06 -- 「← 戻る」クリック --> S03

    S07_MODAL -- 「保存」クリック --> S05
    S07_MODAL -- 「× 閉じる / キャンセル」クリック --> S05

    S08 -- @username / アバタークリック --> S05
```

---
