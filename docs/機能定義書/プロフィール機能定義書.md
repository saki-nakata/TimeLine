# プロフィール機能定義書

**バージョン:** 1.0
**作成日:** 2026-05-17
**作成者:** Nakata Saki

---

## 1. 機能概要

ログイン済みユーザーが他のユーザーのプロフィールを閲覧し、自分自身のプロフィール（アイコン・自己紹介）を編集できる機能。
プロフィール画面（S-05）では、対象ユーザーの基本情報・投稿一覧・フォロー状態を確認できる。
プロフィール編集画面（S-07）では、自分のアイコン画像と自己紹介文を更新できる。

---

## 2. ユーザーストーリー

| ID | ストーリー |
|----|---------|
| US-PROF-01 | ログイン済みユーザーとして、他のユーザーのプロフィール画面でアイコン・自己紹介・フォロワー数・投稿一覧を確認したい |
| US-PROF-02 | ログイン済みユーザーとして、自分のプロフィール画面から「プロフィールを編集」ボタンを押して編集画面に遷移したい |
| US-PROF-03 | ログイン済みユーザーとして、プロフィール編集画面でアイコン画像を変更したい |
| US-PROF-04 | ログイン済みユーザーとして、プロフィール編集画面で自己紹介文を入力・更新したい |

---

## 3. 機能詳細

### 3.1 プロフィール表示

| 項目 | 内容 |
|------|------|
| 表示場所 | プロフィール画面（S-05） |
| URL | `/profile/:userId` |
| 表示内容 | アイコン画像 / @username / 自己紹介（bio）/ フォロワー数 / フォロー中数 / 投稿一覧 |
| アイコン | 未設定時はデフォルト画像を表示 |
| 自己紹介 | 未設定時は非表示（空欄扱い） |
| 投稿一覧 | 対象ユーザーの投稿を新着順で表示 |
| フォローボタン | 自分自身のプロフィールには表示しない。他ユーザーにはフォロー状態に応じて表示 |
| 編集ボタン | 自分自身のプロフィール画面にのみ「プロフィールを編集」ボタンを表示 |

### 3.2 プロフィール編集

| 項目 | 内容 |
|------|------|
| 表示場所 | プロフィール編集画面（S-07） |
| URL | `/profile/edit` |
| アクセス可能者 | 自分自身のみ |
| アイコン変更 | 画像ファイルを選択してアップロード（S3 に保存）。現在のアイコンをプレビュー表示 |
| 自己紹介編集 | テキストエリアで入力。文字数制限は実装時に決定（目安: 160 文字） |
| 保存ボタン | 変更を確定して S-05 に戻る |
| キャンセルボタン | 変更を破棄して S-05 に戻る |

## 4. 使用 API エンドポイント

| Method | エンドポイント | 説明 |
|--------|--------------|------|
| GET | `/api/users/{id}` | プロフィール取得（フォロー状態・フォロワー数を含む） |
| GET | `/api/users/{id}/posts` | ユーザー投稿一覧取得（新着順） |
| PUT | `/api/users/me` | 自分のプロフィール更新（アイコン・bio） |

### レスポンス例（GET /api/users/{id}）

```json
{
  "id": 1,
  "username": "alice",
  "avatarUrl": "https://s3.example.com/avatars/alice.jpg",
  "bio": "Web開発が好きです。",
  "followerCount": 42,
  "followingCount": 18,
  "followedByCurrentUser": false
}
```

## 5. バリデーション仕様

### フロントエンド

| 対象 | ルール | 動作 |
|------|--------|------|
| アイコン画像 | JPEG / PNG / GIF のみ | 「JPEG・PNG・GIF 形式のファイルを選択してください」 |
| アイコン画像 | 4MB 以下 | 「ファイルサイズは 4MB 以下にしてください」 |
| 自己紹介 | 任意入力（空欄可） | — |

### バックエンド

| 対象 | ルール | HTTP ステータス |
|------|--------|---------------|
| 認証 | JWT が有効であること | 401 Unauthorized |
| 編集権限 | 自分自身のプロフィールのみ編集可能 | 403 Forbidden |
| アイコン MIME タイプ | JPEG / PNG / GIF | 400 Bad Request |
| アイコンサイズ | 4MB 以下 | 400 Bad Request |

---

## 6. 制約・前提条件

| 分類 | 内容 |
|------|------|
| 認証 | プロフィール編集はログイン済みユーザーのみ可能 |
| 閲覧 | プロフィール表示はログイン済みユーザーのみ可能 |
| 編集対象 | 自分のプロフィールのみ編集可能（他ユーザーの編集は不可） |
| アイコン保存 | S3 に保存し、`users.avatar_url` に URL を記録する |

---

## 7. 対象外仕様

| 対象外機能 | 理由 |
|-----------|------|
| ユーザー名の変更 | スコープ外 |
| メールアドレスの変更 | スコープ外 |
| パスワード変更 | スコープ外 |
| カバー画像 | スコープ外 |
