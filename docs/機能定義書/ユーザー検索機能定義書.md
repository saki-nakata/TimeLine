# ユーザー検索機能定義書

**バージョン:** 1.0
**作成日:** 2026-05-17
**作成者:** Nakata Saki

---

## 1. 機能概要

ログイン済みユーザーがユーザー名（@username）または表示名で他のユーザーを検索できる機能。
ナビゲーションの検索アイコンをクリック/タップすると SearchOverlay（S-06）がインライン表示され、入力に応じてドロップダウンで一致するユーザーを表示する。専用ページへの遷移は行わない。

---

## 2. ユーザーストーリー

| ID | ストーリー |
|----|---------|
| US-SRCH-01 | ログイン済みユーザーとして、サイドバーの検索バーにユーザー名を入力して検索したい |
| US-SRCH-02 | ログイン済みユーザーとして、検索候補に表示されたユーザーをクリックして、そのユーザーのプロフィール画面に遷移したい |

---

## 3. 機能詳細

### 3.1 検索操作

| 項目 | 内容 |
|------|------|
| 検索 UI | SearchOverlay（ナビゲーションの検索アイコンをクリック/タップするとトグル表示） |
| 検索キー | ユーザー名（`users.username`）または表示名（`users.display_name`）での部分一致 |
| 検索方法 | 検索バーに入力すると 300ms debounce 後に自動検索。結果はドロップダウンで表示 |
| URL | API: `GET /api/users/search?q=xxx`（専用検索ページへの遷移なし） |

### 3.2 検索結果の表示

| 表示項目 | 内容 |
|---------|------|
| アバター | アバター画像（未設定時はデフォルト画像） |
| @username | ユーザー名。クリックでプロフィール画面（S-05）へ遷移 |
| アバター | アバター画像（未設定時はイニシャル表示） |
| 表示名 | `display_name`（未設定時は `username`） |
| @username | ユーザー名 |

### 3.3 検索結果が 0 件の場合

「見つかりませんでした」のメッセージを表示する。

### 3.4 検索ワードが空の場合

ドロップダウンを非表示にし、API 呼び出しは行わない。

---

## 4. 使用 API エンドポイント

| Method | エンドポイント | パラメータ | 説明 |
|--------|--------------|-----------|------|
| GET | `/api/users/search` | `q=xxx` | ユーザー名で部分一致検索 |

### レスポンス例

```json
[
  {
    "id": 3,
    "username": "alice",
    "avatarUrl": null,
    "followerCount": 42,
    "followingCount": 18,
    "followedByCurrentUser": false
  }
]
```

---

## 5. バリデーション仕様

### フロントエンド

| 対象 | ルール | 動作 |
|------|--------|------|
| 検索ワード | 空欄の場合は API 呼び出しを行わない | ドロップダウン非表示 |

### バックエンド

| 対象 | ルール | HTTP ステータス |
|------|--------|---------------|
| 認証 | JWT が有効であること | 401 Unauthorized |
| クエリパラメータ `q` | 必須（空文字不可） | 400 Bad Request |

---

## 6. 制約・前提条件

| 分類 | 内容 |
|------|------|
| 認証 | ログイン済みユーザーのみ検索可能 |
| 検索対象 | ユーザー名（`username`）と表示名（`display_name`）。メールアドレスは検索対象外 |
| 検索方式 | 部分一致（SQL の `ILIKE '%xxx%'`、大文字小文字を区別しない） |
| 表示件数 | 最大 20 件 |

---

## 7. 対象外仕様

| 対象外機能 | 理由 |
|-----------|------|
| メールアドレスによる検索 | プライバシー配慮 |
| 投稿内容・コメントの検索 | スコープ外 |
| 検索履歴の保存 | スコープ外 |
| オートコンプリート / サジェスト | スコープ外 |
