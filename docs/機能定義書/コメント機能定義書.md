# コメント機能定義書

**バージョン:** 1.1
**作成日:** 2026-05-17
**更新日:** 2026-05-22
**作成者:** Nakata Saki

---

## 1. 機能概要

ログイン済みユーザーが任意の投稿にコメントを追加・閲覧・削除できる機能。
コメントは投稿詳細画面（S-04）に時系列順で表示する。
コメント数は投稿カードに常時表示する。

---

## 2. ユーザーストーリー

| ID | ストーリー |
|----|---------|
| US-CMT-01 | ログイン済みユーザーとして、投稿詳細画面のコメント入力欄にテキストを入力して「コメントする」ボタンを押し、コメントを追加したい |
| US-CMT-02 | ログイン済みユーザーとして、投稿詳細画面でその投稿に対するコメント一覧を時系列順に確認したい |
| US-CMT-03 | ログイン済みユーザーとして、自分のコメントの横に表示される削除ボタンから、コメントを削除したい |
| US-CMT-04 | ログイン済みユーザーとして、タイムラインの投稿カードに表示されるコメント数を見て、どれくらい反応があるか確認したい |

---

## 3. 機能詳細

### 3.1 コメント投稿

| 項目 | 内容 |
|------|------|
| 入力フォーム設置場所 | コメントモーダル（投稿カードのコメントボタンで開く）の下部 |
| テキスト入力 | テキストエリア。最大 280 文字（投稿と同じ上限） |
| 「コメントする」ボタン | テキストが 1 文字以上入力されている場合にのみ活性化 |
| 投稿後の動作 | コメント一覧の末尾に即時追加。コメント数カウンターも +1 更新 |

### 3.2 コメント一覧表示

| 項目 | 内容 |
|------|------|
| 表示場所 | コメントモーダル（タイムラインのコメントボタンから開くモーダル） |
| 並び順 | 時系列順（古いコメントが上・新しいコメントが下） |
| 表示項目 | @username・コメント日時（相対表示）・コメントテキスト |
| @username | クリックでプロフィール画面（S-05）へ遷移 |

### 3.3 コメント数の表示

| 項目 | 内容 |
|------|------|
| 表示場所 | タイムライン（S-03）の投稿カード・投稿詳細画面（S-04） |
| データ源 | `posts.comment_count`（非正規化カウンタ） |
| 更新タイミング | コメント追加・削除後にリアルタイム反映 |

### 3.4 コメント削除

| 項目 | 内容 |
|------|------|
| 表示条件 | 削除ボタンは自分のコメントにのみ表示 |
| 削除確認 | 確認ダイアログなしで即時削除 |
| 削除後の動作 | コメント一覧から除去。コメント数カウンターも -1 更新 |

---

## 4. 使用 API エンドポイント

| Method | エンドポイント | 説明 |
|--------|--------------|------|
| GET | `/api/posts/{id}/comments` | コメント一覧取得（時系列順） |
| POST | `/api/posts/{id}/comments` | コメント投稿 |
| DELETE | `/api/posts/{id}/comments/{commentId}` | コメント削除（自分のコメントのみ） |

---

## 5. バリデーション仕様

### フロントエンド

| 対象 | ルール | エラーメッセージ |
|------|--------|----------------|
| コメントテキスト | 必須（1 文字以上） | 「コメント内容を入力してください」 |

### バックエンド

| 対象 | ルール | HTTP ステータス |
|------|--------|---------------|
| コメントテキスト | 必須（1〜280 文字） | 400 Bad Request |
| 認証 | JWT が有効であること | 401 Unauthorized |
| 削除権限 | 自分のコメントのみ削除可能 | 403 Forbidden |
| 存在確認 | 対象投稿・コメントが存在すること | 404 Not Found |

---

## 6. 制約・前提条件

| 分類 | 内容 |
|------|------|
| 認証 | コメント投稿・削除はログイン済みユーザーのみ可能 |
| 閲覧 | コメント一覧はログイン済みユーザーのみ閲覧可能 |
| カウンタ更新 | `posts.comment_count` はサービス層で `+1/-1` 更新する |

---

## 7. 対象外仕様

| 対象外機能 | 理由 |
|-----------|------|
| コメントへの返信（ネスト） | スコープ外 |
| コメントの編集 | スコープ外 |
| コメントへのいいね | スコープ外 |
